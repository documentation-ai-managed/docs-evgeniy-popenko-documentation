## Recent Releases

Review the most recent versions below. You access detailed release notes, migration guides, and contribution instructions within each section.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added support for custom brand colors, like `#3B82F6` for your projects
- Introduced real-time collaboration previews in the editor

## Improvements

- Enhanced MDX parsing speed by 40% for larger documentation sites
- Optimized image loading with automatic lazy-loading

## Bug Fixes

- Fixed nested code block rendering issues in JSX components
- Resolved frontmatter parsing errors with blank lines

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated Lucide icon library with 200+ searchable icons
- Added `<Video>` and `<Iframe>` components for rich media embeds

## Improvements

- Improved accessibility with required `alt` and `title` attributes
- Streamlined component syntax rules in the documentation

## Bug Fixes

- Corrected `<ParamField>` location attribute validation (one only)
- Patched Mermaid diagram code block rendering (no JSX component)

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["breaking", "feature"]}>

## Breaking Changes

- Removed legacy `<Mermaid>` JSX component; use code blocks only

## New Features

- Launched core MDX components: `<Steps>`, `<Tabs>`, `<Update>`, `<Columns>`
- Enabled YAML frontmatter validation without blank lines

## Improvements

- Added parsing error prevention checklist for writers
- Introduced 5-8 component diversity requirement per page

</Update>
